# macOS 27 Theme for Squirrel (Rime)

English | [简体中文](#简体中文)

A green theme for Squirrel (Rime) inspired by the native macOS 27 input method.

## ✨ Features

- 🍃 Inspired by the native macOS 27 input method
- 🟢 macOS-style green highlight
- 🌗 Automatic Light / Dark mode support
- 🎨 Display P3 color space
- ✨ Optimized spacing, typography, transparency and corner radius
- 💻 Designed to better match the overall visual style of macOS

## 📦 Installation

1. Copy `squirrel.custom.yaml` to:

```text
~/Library/Rime/
```

2. Merge it with your existing configuration if necessary.

3. Click **Redeploy** from the Squirrel menu.

# 简体中文

一个参考 **macOS 27 原生输入法**设计风格的鼠须管（Squirrel / Rime）主题。

## ✨ 特性

- 🍃 参考 macOS 27 原生输入法设计
- 🟢 原生风格绿色高亮
- 🌗 支持浅色 / 深色模式自动切换
- 🎨 使用 Display P3 色彩空间
- ✨ 优化圆角、字体、透明度、间距等细节
- 💻 更贴近 macOS 系统整体视觉风格

## 📦 安装

1. 将 `squirrel.custom.yaml` 复制到：

```text
~/Library/Rime/
```

2. 如已有 `squirrel.custom.yaml`，请将配置合并到现有文件中。

3. 点击菜单栏鼠须管图标，选择 **重新部署（Redeploy）** 即可生效。
