# MacOS 27 Theme for Squirrel（Rime）

一个参考 **macOS 27 原生输入法**设计风格的鼠须管（Squirrel / Rime）主题。

## ✨ 特性

- 🍃 参考 macOS 27 原生输入法设计
- 🟢 原生风格绿色高亮
- 🌗 支持浅色 / 深色模式自动切换
- 🎨 使用 Display P3 色彩空间
- ✨ 优化圆角、字体、间距等细节
- 💻 更贴近 macOS 系统整体视觉风格

## 📦 安装

将 `squirrel.custom.yaml` 复制到：

```text
~/Library/Rime/
```

然后点击菜单栏鼠须管图标，选择 **重新部署（Redeploy）** 即可生效。

# MacOS 27 Theme for Squirrel（Rime）

A macOS 27-inspired green theme for Squirrel (Rime).

## Features

- 🍃 Inspired by the native macOS 27 input method
- 🟢 macOS-style green highlight
- 🌗 Light & Dark mode support
- 🎨 Display P3 color space
- ✨ Optimized spacing, typography and corner radius

## Installation


1. Copy `squirrel.custom.yaml` to:

```text
~/Library/Rime/
```

2. Replace it with your existing configuration.

3. Click **Deploy** from the Squirrel menu.
